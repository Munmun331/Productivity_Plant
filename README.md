# 🌱 Productivity Plant

## 📌 Overview
Productivity Plant is a console-based productivity application that helps users stay focused and organized by combining task management with Pomodoro-based study sessions. It gamifies productivity by rewarding users with points and achievements, encouraging consistent progress.

---

## ⚙️ Features
- ✅ To-Do List Management (Add, View, Complete Tasks)  
- ⏳ Pomodoro Study Sessions with customizable durations  
- 📚 Normal Study Mode for flexible sessions  
- 🏆 Achievement Tracking (Pomodoro & Study Count)  
- 🔐 User Authentication (Login & Account Creation)  
- 💾 Data Persistence using file handling  
- 🧮 Built-in Calculator and BMI Calculator  

---

## 🛠️ Technologies Used
- C++ (OOP Concepts)
- File Handling (Data Storage)
